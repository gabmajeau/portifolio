Proyecto de UX – Vínculo afectivo con el usuario

HTML/CSS y JavaScript

Este fue un proyecto de prácticas que forma parte de una investigación en la que exploramos el campo del UX como una herramienta estratégica para crear un vínculo afectivo con el usuario.

Se incorporaron elementos de los años 2000, basándose principalmente en el clásico sistema operativo Windows XP, donde los íconos generan estímulos nostálgicos, como Orkut, Windows Media Player, sus efectos visuales clásicos y menús de juegos famosos.

Este proyecto tuvo un desarrollo complejo por integrar blockchain, backend e frontend. Yo desarrollé el frontend, donde, utilizando HTML, CSS e Javascript, entregué a tiempo un buen resultado final:

Homenaje a la franquicia Zelda

Al hacer clic en los logotipos, el usuario es redirigido a los videos. Considero que este es mi mejor ejemplo de experiencia de usuario:

Al principio, el proyecto era muy pesado debido a la cantidad de audio y video, pero encontré una solución usando "Cloudinary", subiendo los archivos a la nube y extrayéndolos con URL. Por ahora, el proyecto solo está disponible en portugués.

Juego de plataformas básico

Javascript

Usando principalmente Javascript, creé un "juego de plataformas" donde los bloques se mueven hacia el jugador, quien simplemente salta. La mayor dificultad fue crear la colisión y el HUD, que inciden directamente en la jugabilidad, como finalizar el juego con un botón y cambiar el volumen (usando una etiqueta). Fue un proyecto interesante, donde agregué algunos sonidos, como saltar, ganar, perder y sumar puntos.

Puse un límite de 50 puntos para que no fuera infinito, utilice los colores temáticos del portafolio y todos los sonidos utilizados fueron de la franquicia de Mario.

Optimización de la página web

Lazy Loading

Neste proyecto yo crié una página de imágenes de alta resolución y, utilizando las DevTools del Google Chrome, simulé una conexión 4G lenta, para mostrar la implementación del lazy loading.

Este setup demonstra cómo el lazy loading genera un delay, disminuyendo el tiempo de cargamento inicial y ahorrando internet para conexiones lentas.
